Linux Auto-Update

Automatic package updates for Linux using systemd timers.

Features

  • Automatic Updates Set a time and your system updates daily.
  • CLI & GUI CLI for servers, zenity GUI for desktops.
  • Multi-Distro Works with yay, pacman, apt, dnf, yum, zypper.
  • Systemd Timers Modern, reliable scheduling.

Installation

  1. Quick Install 
    bash -c "$(curl -fsSL https://raw.githubusercontent.com/PlayWitIt/linux-auto-update/main/setup_autoupdate.sh)"
  2. Or Manual Install 
    curl -fsSL https://raw.githubusercontent.com/PlayWitIt/linux-auto-update/main/setup_autoupdate.sh -o setup_autoupdate.sh
chmod +x setup_autoupdate.sh
./setup_autoupdate.sh --install --time 07:30

CLI Usage (Server/Headless)

# Install with daily update at 7:30 AM
./setup_autoupdate.sh --install --time 07:30

# Check status
./setup_autoupdate.sh --status

# Remove service
./setup_autoupdate.sh --remove

# Grant passwordless sudo
./setup_autoupdate.sh --grant-sudo

# Show help
./setup_autoupdate.sh --help

GUI Mode (Desktop)

Run without arguments or with --gui (requires zenity):

./setup_autoupdate.sh
# or
./setup_autoupdate.sh --gui
